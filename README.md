# Simple Interest Calculator

A simple command-line or script-based tool to calculate simple interest based on principal amount, annual interest rate, and time period.

## Formula
The simple interest is calculated using the standard formula:
A = P * R * T / 100

Where:
- `P` = Principal amount
- `R` = Annual interest rate (in percentage)
- `T` = Time period (in years)

## How to Use
1. Clone or download this repository.
2. Input your principal amount, rate of interest, and time period.
3. Run the script to calculate the total interest accrued over the given timeframe.
